Tecnología

Instructivo:

1. El juego se acaba cuando completas 3 puzzles.

2. Para completar un puzzle tienes que llenar todas las celdas que se te piden.

3. Mientras más rápido completes el puzzle, más puntos tendrás.

4. Para completar el puzzle, tienes que seguir los indicadores de las filas y columnas.

5. Si el indicador se encuentra de color amarillo es porque se cumplió el patrón requerido, de lo contrario se mostrará blanco (que el indicador sea de color amarillo no significa que las celdas estén en la posición correcta)

Ingeniería

Instructivo:

1. Existen 3 niveles y cada uno se reinicia cuando mueres.

2. Existen 5 preguntas de razonamiento numérico y 10 de razonamiento abstracto. Cada competencia se califica sobre 100 puntos y cada pregunta vale lo mismo.

3. Recoge todas las monedas y diamantes para poder desbloquear los problemas. Encuentra la palanca, gírala y los verás.

4. Para completar el puzzle, tienes que seguir los indicadores de las filas y columnas.

5. Si el indicador se encuentra de color amarillo es porque se cumplió el patrón requerido, de lo contrario se mostrará blanco (que el indicador sea de color amarillo no significa que las celdas estén en la posición correcta)

Arte

Instructivo:

1. El juego se acaba después de completar exitosamente 3 cuadros.

2. Entre más rápido completes los cuadros mayor puntuación tendrás.

3. Deberás generar los colores solicitados recolectando las gotas que caen.

4. En la esquina superior izquierda podrás observar el color generado o recolectado.

5. En la esquina superior derecha podrás observar los colores que debes juntar para completar el cuadro.

6. Cuando tu color actual coincide con alguno de los colores solicitados, corre hacia el cuadro y colócalo.

Combinación de colores

Rojo = Rojo

Azul = Azul

Amarillo = Amarillo

Negro = Negro

Blanco = Blanco

Blanco + Negro = Gris

Amarillo + Rojo = Naranja

Azul + Amarillo = Verde

Azul + Amarillo + Rojo = Café

Azul + Rojo = Violeta

Azul + Rojo + Azul = Azul Violaceo

Controles

Lanzar color al cuadro: Espacio

Movimiento: A, D

Pausa: ESC
